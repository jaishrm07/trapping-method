# Paper Notes

One MD per paper, named `<lastname><year>_<slug>.md`. Examples:
- `finn2017_maml.md`
- `lermen2024_lora_undoes_safety.md`
- `sinha2018_dro.md`

## Note template

```markdown
# <full title>

**Authors / venue / year**:
**Bib key**:
**Read for thread(s)**: 01, 02, ...
**Read on**: YYYY-MM-DD

## TL;DR (3 sentences)

## Why we read it

What question from the threads/ files is this answering?

## Key claims (with location)

1. <claim> (Eq./Sec. X.Y)

## Methods we could borrow / discard

## Open questions / disagreements

## Citation
```

Keep notes terse. The goal is to extract what's useful for our threads,
not summarize the paper for general consumption.
